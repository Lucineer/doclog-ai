# DocLog — Living Codebase Documentation

A repo-native AI that watches your codebase evolve and auto-updates docs. After 6 months, it knows your API better than any new hire.

## Features
- 📡 API endpoint registry with auth, schemas, deprecation
- 🏛️ Architecture Decision Records (ADRs)
- 📝 Auto-generated changelogs per version
- 🔍 Semantic search across all documentation
- ⚠️ Staleness detection for outdated docs
- 🧠 AI that answers questions with specific doc references

## The Cocapn Difference
Every endpoint, every ADR, every breaking change is accumulated in the repo. A new hire asks DocLog a question and gets an answer that references the specific ADR where the decision was made. No other tool has this accumulated context.

## Deploy
1. Fork this repo
2. Set DEEPSEEK_API_KEY
3. Deploy to Cloudflare Workers
